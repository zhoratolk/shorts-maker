<objective>
Build the deterministic profanity-detection module: a new `scripts/profanity.py` that loads a local RU+EN swear wordlist, normalizes each Whisper word token for common obfuscated spellings, matches word-boundary stems, and returns clip-relative masked spans (padded, merged, span-capped). Ships the committed `data/profanity_wordlist.yaml` data file and the mirrored `tests/test_profanity.py`.

Purpose: This is the AUDIO-01 layer ‚Äî "swear words are identified from the existing word-level Whisper transcript for every clip before render." It is a pure downstream consumer of the existing transcript; no new transcription work. Detection is mechanical/deterministic (D-02, MONET-02 precedent), so it lives in a script, not the orchestrator.
Output: `scripts/profanity.py` (`load_wordlist`, `normalize_word`, `compile_patterns`, `find_profane_spans`, argparse `main()` CLI), `data/profanity_wordlist.yaml` (committed, generic), `tests/test_profanity.py`.
</objective>
